NOMOS

High-Performance C++ Web Framework

Modern C++ • Blazing Fast • Minimalist

About the Project

Nomos is a lightweight, ultra-fast web framework built in modern C++. It focuses on low latency, efficient memory management, and a clean API design for developers who require bare-metal routing speed and structural control.

Key Features

High Performance

Designed from the ground up to minimize overhead and handle thousands of concurrent requests smoothly.

Modern C++ Architecture

Leverages robust compile-time optimizations and type safety inherent in modern standards.

Clean Routing

Simple, intuitive syntax for defining endpoints, handling requests, and processing responses.

Quick Start

#include "nomos.hpp"

int main() {
    Nomos::App app;

    app.route("/", [](const auto& req, auto& res) {
        res.send("Hello from Nomos!");
    });

    app.listen(8080);
    return 0;
}
